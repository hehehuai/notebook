# 常用操作#

| 操作             | 快捷键      |
|------------------|-------------|
| 打开shell        | spc + '     |
| 当前目录搜索文件 | spc + f + f |
| 项目中搜索字符串 | spc + /     |
| 注释一行         | spc + c + l |
